# Dashboard

Das Dashboard ist deine Startseite. Ein Raster aus Kacheln, jede zeigt einen Wert vom Boot.

## Kacheln

Die Standard­anordnung enthält Geschwindigkeit, Kurs, Tiefe, Wind, Position und Batterie — aber alles davon lässt sich anpassen. Kacheltypen:

- **Geschwindigkeit**, **Kurs**, **Tiefe**, **Position** — die wichtigsten Navigationswerte.
- **Wind** und **Wind-Rose** — scheinbarer und wahrer Wind, mit Rosen-Visualisierung.
- **Batterie** und **Batterie-Strom** — Spannung, Ladezustand, Strom rein/raus.
- **Krängung** und **Trimm** — Bootslage.
- **Relais** — einen Stromkreis ein- oder ausschalten und seinen Zustand sehen.
- **Tank** — aktueller Füllstand eines konfigurierten Tanks.
- **Wettervorhersage**-Kacheln — Wellen, Böen, Luftdruck, Wassertemperatur, Lufttemperatur.

Plugins können auch eigene Kacheln beisteuern — sie erscheinen neben den eingebauten in der Auswahl.

Tippe eine Kachel an, um die **Detailansicht** mit Verlaufsdiagrammen und Statistiken zu öffnen.

## Bearbeitungs-Modus

Über das **Stift**-Symbol in der Seitenleiste startet der Bearbeitungs-Modus. In diesem Modus:

- **Ziehen** zum Verschieben.
- **Größe ändern** durch Ziehen am Eck-Anfasser.
- **Hinzufügen** einer Kachel aus der Auswahl.
- **Entfernen** über das × auf der Kachel.
- **Konfigurieren** einer Relais-Kachel (welches Relais) oder Tank-Kachel (welcher Tank) über das Zahnrad.
- **Zeilenanzahl** mit den ±-Reglern, um mehr oder weniger auf den Bildschirm zu bekommen.
- **Seitenleiste umstellen** durch die vier Kanten (links → rechts → oben → unten → links) über den Positions-Knopf.

Mit **Fertig** schließt du den Modus. Layouts werden **pro Client** gespeichert — Steuerstand und Salon können also völlig unterschiedliche Dashboards haben.

## Seitenleiste

Die Seitenleiste enthält Schnellzugriffe auf Karte, Instrumente-Ansicht, Relais-Ansicht, Bearbeitungs-Modus, Einstellungen und Hilfe. Sie kann an jeder Bildschirmkante sitzen — die Position wird über den Knopf im Bearbeitungs-Modus gewechselt. Auf schmalen Bildschirmen klappt sie automatisch auf reine Symbole zusammen.

Im **Karten-Modus** (Einstellungen → Karte) ersetzt die Karte das Dashboard als Startseite, und die Dashboard-Seitenleiste ist nicht sichtbar. Die Hilfe ist dann von innerhalb der Einstellungen erreichbar.
