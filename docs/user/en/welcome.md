# Welcome to BigaOS

BigaOS is a marine navigation and vessel-monitoring system designed for boats. The core software reads no data on its own — instead it loads **plugins** that connect to your boat. Whatever a plugin produces (speed, heading, depth, wind, position, batteries, tanks, engine…) BigaOS displays on as many screens and phones as you like.

Browse **Settings → Plugins** for drivers covering different ways to get data into your server — pick the ones that match your hardware. Each plugin ships its own help articles in the sidebar so you can read up before installing.

## What you can do

- **Navigate** with the chart view. Plot routes that stay in water, follow them with the autopilot, drop a marker with a long-press, and search for places by name.
- **Monitor** every instrument from any display or phone connected to the boat's server.
- **Customize** dashboards per display so the helm shows what the helm needs and the salon shows what the salon needs.
- **Get alerted** for wind, waves, depth, batteries — both from live sensor data and from the weather forecast.
- **Drop anchor** with a chain-length recommendation calculated from your vessel's actual dimensions and chain spec, then activate the swing-radius alarm.
- **Control switches** wired through the boat's electrical system, right from a dashboard tile or a physical button on a Pi panel.
- **Read tank levels** with a calibration wizard for any shape of tank.
- **Console access** to the server itself — live logs and an interactive shell, for when something needs poking at.
- **Extend** the system with plugins for new data sources, languages, and dashboard tiles.
