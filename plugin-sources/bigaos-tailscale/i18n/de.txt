tailscale.setup_title=Einrichtung
tailscale.setup_instructions=1. Erstelle ein kostenloses Tailscale-Konto auf tailscale.com\n2. Gehe in der Admin-Konsole zu Settings > Keys und erstelle einen Auth Key ("Reusable" fuer Headless-Geraete aktivieren)\n3. MagicDNS aktivieren — Admin-Konsole > DNS > "MagicDNS" einschalten. So erreicht jedes Geraet im Tailnet das Boot ueber den Maschinennamen statt einer IP-Adresse.\n4. Fuege den Key unten ein und bestaetige den Hostnamen (Standard: bigaos)\n5. Auf dem Handy (mit Tailscale installiert und verbunden) die BigaOS-App oeffnen und mit http://<hostname>:3000 verbinden — zum Beispiel http://bigaos:3000. Der Hostname bleibt stabil, auch wenn sich IPs aendern.\n6. Bei Verwendung von Subnet-Routing muessen die Routen in der Tailscale-Admin-Konsole unter den Maschineneinstellungen genehmigt werden.
tailscale.auth_key=Auth Key
tailscale.auth_key_desc=Dein Tailscale Auth Key (beginnt mit "tskey-auth-")
tailscale.hostname=Hostname
tailscale.hostname_desc=Name dieses Bootes im Tailscale-Netzwerk. Mit aktiviertem MagicDNS erreicht dein Handy den Server unter http://<hostname>:3000 — diesen Wert in der BigaOS-App statt einer IP eintragen.
tailscale.subnet_routes=Subnetz-Routen
tailscale.subnet_routes_desc=Lokale Netzwerkrouten bekannt geben (automatisch erkannt wenn leer). Beispiel: 192.168.1.0/24
tailscale.accept_routes=Routen akzeptieren
tailscale.accept_routes_desc=Routen von anderen Tailscale-Geraeten akzeptieren
